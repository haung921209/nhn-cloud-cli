# NHN Cloud CLI Use Cases

This document serves as an index for various service scenarios and guides.

## Service Guides

### [Compute & Network](guides/COMPUTE_USE_CASES.md)
Managed compute instances, SSH key management, and network configurations.
- **[Instance Lifecycle](guides/COMPUTE_USE_CASES.md#1-instance-lifecycle)**: Create, List, Start, Stop.
- **[SSH Connect](guides/COMPUTE_USE_CASES.md#4-compute-access-ssh)**: Connect to instances using the `connect` command.
- **[Key Import](guides/COMPUTE_USE_CASES.md#prerequisite-import-key)**: Securely import existing private keys.

### [RDS](guides/RDS_USE_CASES.md)
Managed Database Services for MySQL, MariaDB, and PostgreSQL.
- **[MySQL Lifecycle](guides/RDS_USE_CASES.md#1-mysql)**: Create, Configure Security Groups, Connect.
- **[MariaDB Lifecycle](guides/RDS_USE_CASES.md#2-mariadb)**: Create, Configure Network, Connect.
- **[PostgreSQL Lifecycle](guides/RDS_USE_CASES.md#3-postgresql)**: Create, Manage specialized Security Groups, Connect.

### [NKS (Kubernetes)](guides/NKS_USE_CASES.md)
Managed Kubernetes Service.
- **[Cluster Lifecycle](guides/NKS_USE_CASES.md#end-to-end-lifecycle)**: Create Cluster, Add Node Groups, Get Kubeconfig.
